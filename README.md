# Game Portfolio Website

Trang web tĩnh để showcase portfolio các tựa game của bạn và nhóm bạn.

## Công nghệ sử dụng

- React 18
- Vite
- CSS3

## Cài đặt

1. Cài đặt dependencies:
```bash
npm install
```

2. Chạy development server:
```bash
npm run dev
```

3. Build cho production:
```bash
npm run build
```

4. Preview build:
```bash
npm run preview
```

## Cấu trúc dự án

```
src/
  ├── components/
  │   └── HomePage.jsx    # Trang chủ
  ├── styles/
  │   └── HomePage.css    # Styling cho trang chủ
  ├── App.jsx
  ├── App.css
  ├── main.jsx
  └── index.css
```
